## 📊 데이터와 분석 방법론

### 🗂️ **데이터 개요**
- **출처**: 통계청 국가통계포털(KOSIS)
- **기간**: 1960년 ~ 2072년 (기존 통계청 예측 데이터)
- **확장**: 2172년까지 150년 장기 예측
- **규모**: 21개 변수 × 114년간 데이터

**주요 분석 변수:**
- 연령별 인구 구성비 (0-14세, 15-64세, 65세 이상)
- 성별 인구 및 성비
- 부양비 (총부양비, 유소년부양비, 노년부양비)
- 노령화지수, 중위연령, 평균연령

### 🔬 **분석 방법론의 체계적 접근**

**1단계: 시계열 분석 (ARIMA & ETS)**
```r
# ARIMA 모델링
auto.arima() # 최적 파라미터 자동 선정
forecast(h = 100) # 100년 예측

# ETS 모델링  
ets() # 지수평활법 기반 예측
```

**2단계: 머신러닝 접근 (Random Forest & LASSO)**
```r
# Random Forest
randomForest(총인구 ~ ., data = train_data)

# LASSO Regression
cv.glmnet(X, y, alpha = 1) # 변수 선택과 예측
```

**3단계: 모델 앙상블**
- 각 방법론의 MAPE 비교를 통한 최적 모델 선택
- ARIMA vs ETS: 변수별 성능 비교 후 선택적 적용
- Random Forest vs LASSO: 앙상블을 통한 성능 개선

## 📈 주요 분석 결과

### 🎯 **시계열 모델 성능**
**ARIMA & ETS 조합 결과:**
- **평균 MAPE**: 0.33217 (33.21% 오차)
- **최고 성능 변수**: 여자명, 성비 등
- **개선 사항**: "인구성장률" 변수 제외로 안정성 확보

**모델별 선택 결과:**
- **ARIMA 선택**: 17개 변수 (대부분의 변수에서 우수한 성능)
- **ETS 선택**: 2개 변수 (성비, 유소년부양비에서 우수)

### 🤖 **머신러닝 모델 성능**
**타겟 변수: 총인구 예측**

| 모델 | MAE | 특징 |
|------|-----|------|
| Random Forest | 143,652 | 안정적 예측, 3,500만명 수준 유지 |
| LASSO | 178,616 | 극단적 인구 감소 예측 |
| **앙상블 모델** | **110,040** | **최고 성능, 균형잡힌 예측** |

### 📊 **변수 중요도 분석**
**Random Forest 기준:**
1. **여자(명)** - 가장 높은 예측력
2. **남자(명)** - 두 번째 높은 예측력  
3. 구성비(%) 15-64세
4. 인구(명) 0-14세
5. 평균연령(세)

**LASSO 기준:**
- 대부분 변수 계수가 0으로 축소
- **남자(명)**: 계수 1.23
- **여자(명)**: 계수 0.71
- 극도로 단순화된 모델
